2024 年，我们在聊天窗口中与 Claude 交互。你提问，得到答案，但需要自己将答案转化为有用的东西。2025 年，Claude Code 让工程师以令人羡慕的速度交付。今年，借助 Claude Cowork，我们所有人都能跟上步伐。

In 2024, we had Claude in a chat window. In 2025, Claude Code let engineers ship at a pace that made the rest of us a little jealous. This year, we can all catch up with Claude Cowork.

现在我 90% 的工作都在 Claude Cowork 中完成。在这篇文章中，我将展示如何判断哪些任务适合放在这里，通过我自己工作中的真实案例进行讲解，并让你在大约十分钟内完成第一个可交付成果。

Now, 90% of my work happens in Claude Cowork. In this post, I'll show you how to tell which of your tasks belong there, walk through real examples from my own work, and get you to your first finished deliverable in about ten minutes.

## 使用 Chat vs Claude Cowork vs Claude Code

如果你的工作是非技术性知识工作——邮件、演示文稿、电子表格、文档、会议以及"你能整理一下这个的摘要吗"——那么 Claude Cowork 适合你。你不需要知道如何编码，也不需要知道什么是"agent"或如何构建 agent。

If your job is non-technical knowledge work – emails, decks, spreadsheets, docs, meetings, and "can you pull together a summary of this" – then Claude Cowork is for you. You don't need to know how to code.

**Chat** 通常是知识工作者认识 Claude 的方式。Chat 用于获取答案、头脑风暴和思考问题。

**Claude Cowork** 将这种模式反转：不是把你的工作带给 Claude，而是把 Claude 带到你的工作中。你把它指向电脑上的一个文件夹，连接到你已经使用的应用，告诉它你想要什么结果，然后离开，回来时工作就完成了。

**Claude Code** 专为构建和交付软件的开发者设计。如果你的工作在代码中，从那里开始。

## 何时使用 Claude Cowork？/ When should you use Claude Cowork?

如果所需内容只需几次交流就能完成（如问题、解释、头脑风暴或直觉验证），使用 chat。

Use chat if what you want fits in a few exchanges, like a question, an explanation, a brainstorm, or a gut check.

如果你需要的是可交付成果（如别人会打开的文件、别人会展示的演示文稿、需要排序的电子表格），使用 Claude Cowork。用于任何多步骤、涉及多个文件/类型或涉及多个应用的任务。

Use Claude Cowork if what you need is a deliverable, for example, a file someone will open, a deck someone will present, or a spreadsheet to be sorted. Use it for anything that's multi-step or touches more than one file.

最常见的错误是使用 chat 处理所有事情，从未体验过 Claude Cowork 的差异。相反的极端错误是用 Claude Cowork 处理一次性问题，然后等待 chat 本可以在五秒内回答的内容。

The most common mistake is reaching for chat for everything and never feeling the difference Claude Cowork can make. The opposite mistake is using Claude Cowork for one-off questions.

## Claude Cowork 任务的五个要素 / The five ingredients of a Claude Cowork-shaped task

不确定哪些项目可以委派给 Claude Cowork？用这个清单检查一下：

1. **多个输入**：多个文件、整个文件夹、或文件加连接器。
2. **输出文件**：需要可交付的成果——文档、演示文稿、电子表格或 CSV。
3. **会重复做**：重复性任务是甜点区，可以安排在你到工位之前运行。
4. **已经知道好结果的样子**：你熟悉输出的形态，能在 15 秒内判断是否正确。
5. **中间部分是无聊的**：思考在开始和结束，中间的所有工作（提取、编译、核对、重新格式化）都可以交给 Claude。

## 我在 Anthropic 如何使用 Claude Cowork / How I use Claude Cowork at Anthropic

**每日简报**：我有一个"每日简报"任务，每天早上 6 点运行。Claude Cowork 连接到我的 Slack 和 Gmail，查看未读邮件和关心的频道，分类后生成简短报告。

Daily briefing: A task that runs every morning at 6am. Claude Cowork is connected to my Slack and Gmail, reviews my unread emails and channels, sorts them into buckets, and produces a short report.

**预算跟踪**：用 Claude Cowork 连接 Google Ads 和 Meta Ads，创建实时 HTML 仪表盘，自动拉取每日支出并计算进度。

Budget pacing: Connect to Google Ads and Meta Ads, create a live HTML dashboard that automatically pulls in daily spend and calculates pacing.

**报告**：Claude Cowork 连接到 Google Search Console，获取关心的数据（查询、国家、页面），整合到单个表格中，按模板生成报告。报告曾经每周花费约 30 分钟；现在只需 5 分钟，而且这 5 分钟花在需要我判断的部分。

Reporting: Claude Cowork connected to Google Search Console pulls what I care about and reconciles it into a single sheet. Reporting used to take ~30 minutes a week; now it takes five.

## Claude Cowork 的前 10 分钟 / Your first 10 minutes with Claude Cowork

1. 打开 Claude 桌面应用，切换到 Claude Cowork 标签页。
2. 给 Claude 一些工作材料——拖入几个文件，指向一个文件夹，或连接常用应用。
3. 告诉 Claude 你想要的最终结果。
4. 从你熟悉的真实任务开始。
5. **让 Claude 在开始前先提问**——这是最有用的习惯。在提示中包含："在我们开始之前，把你的理解复述给我确认，然后尽可能多地提出澄清问题。"

## 我仍然使用 Chat 的场景 / When I still reach for chat

我仍然大量使用 chat 来讨论定位问题、在投入之前验证想法，或问一些随机问题。

I still use chat extensively to talk through a positioning problem, pressure-test an idea before I commit to it, or to ask random questions.

关键是，chat 用于当输出是你脑海中的一个想法时，而 Claude Cowork 用于当输出是你将交给别人的东西时。

Chat is for when the output is a thought in your head, and Claude Cowork is for when the output is something you'll hand to someone else.

## 去构建一些东西 / Go build something

选一个你每周做的重复性任务，尝试用 Claude Cowork 处理，看看结果如何。前几次可能有些笨拙，但几次之后，你会很快从"我怎么用这个"变成"我接下来给它什么"。

Pick one repetitive task you do every week, try using Claude Cowork for it, and see what comes back. After a few tries you'll quickly go from "how do I use this" to "what do I hand it next."
