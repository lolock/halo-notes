# Claude Design 现已支持日常工作中的品牌一致性 / Claude Design now stays on brand for daily work

> EN: Claude Design now sticks to your design system across projects, works fluidly with Claude Code, lets you edit directly on the canvas, and connects to more tools you already use. It has a new home in the sidebar on the Claude desktop app; you can also find it at claude.ai/design.

Claude Design 现在能够跨项目遵循你的设计系统，与 Claude Code 流畅配合，让你直接在画布上进行编辑，并连接更多你已经使用的工具。它现在位于 Claude 桌面应用侧边栏的新位置；你也可以在 claude.ai/design 找到它。

我们的目标一如既往：让你在截止日期内探索比以往更多的方向，并使交付打磨或构建的过程尽可能无缝。如今这个目标正在服务更多用户。超过一百万人在第一周内使用了 Claude Design，你们的反馈正在塑造我们下一步的构建方向。

## 用你的设计系统组件构建 / Build with your design system components

> EN: We rebuilt the design system import to give you more flexibility and increase precision. Bring in one or several design systems from a GitHub repo, design files, or raw uploads. Claude builds with your components, checks its output against your design system, and makes corrections before you see it.

我们重新构建了设计系统导入功能，为你提供更多灵活性并提高精度。可以从 GitHub 仓库、设计文件或原始上传中导入一个或多个设计系统。Claude 使用你的组件进行构建，对照你的设计系统检查输出，并在你看到之前进行修正。

对于较大的团队，新增的管理员角色可以批准一个标准系统并锁定编辑权限，确保作品始终符合公司规范。

## 在设计与代码之间切换 / Move between design and code

> EN: It's now easier to move between working in Claude Design and Claude Code while keeping your work synced, with updates rolling out today. Use /design-sync to pull in your design system, so everything you build in Claude Design starts from your existing components.

现在在 Claude Design 和 Claude Code 之间切换工作变得更加便捷，同时保持工作同步，更新已于今日开始推送。使用 `/design-sync` 拉入你的设计系统，这样你在 Claude Design 中构建的一切都从现有组件开始。

当设计准备好成为软件时，你可以将其交接给 Claude Code，它会在现有工作的基础上继续推进，而不是从截图重新开始。

偏好从 Claude Code 开始？你可以不离开终端，使用 `/design` 命令创建、编辑和同步设计项目。将设计导入代码库，将代码转换为实时原型，或让 Claude 将一个项目从头带到尾。

## 日常使用更稳定 / Steadier for daily work

> EN: Our new editor gives you direct, fine-grained control over every element of your designs. New, rich layout controls let you drag, resize, and align elements. Hundreds of stability fixes make the editor hold up under real use.

我们的新编辑器让你可以对设计的每个元素进行直接、精细的控制。新增的丰富布局控制支持拖拽、调整大小和对齐元素。数百项稳定性修复使编辑器在真实使用场景下更加可靠。

Claude Design 现在还与聊天、Claude Cowork 和 Claude Code 共享使用额度，因此大多数人将获得更大的创作空间，遇到额度限制的频率也会降低。平均每次交互使用的 token 数更少，同样效果下效率更高，错误率也大幅下降。

## 快速上手 / Getting started

> EN: Claude Design is in beta on Claude Pro, Max, Team, and Enterprise plans and included with your subscription. It's turned off by default for Enterprise users; admins can enable it in organization settings, and work is shareable only within your organization.

Claude Design 在 Claude Pro、Max、Team 和 Enterprise 套餐中处于 Beta 阶段，包含在你的订阅中。Enterprise 用户默认关闭，管理员可在组织设置中启用，作品仅在组织内部可共享。

在 claude.ai/design 或桌面应用的侧边栏中找到它。导入你的设计系统，开始创作。没有灵感？让你的下一个着陆页尝试三个方向的方案，挑选值得打磨的那一个。
